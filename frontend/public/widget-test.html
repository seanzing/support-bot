External Website Demo

This page demonstrates how the ZING Support Chat widget can be embedded on any external website with just one script tag.

How It Works

The ZING Support widget is embedded on this page using a single script tag. Look at the bottom-right corner of your screen - you'll see the floating chat bubble!

📌 Try it: Click the purple chat bubble in the bottom-right corner, or click the "Open Chat Widget" button above to open the support chat.

Embed Code

Add this to any website to enable ZING Support Chat:

<script src="https://zing-customer-support-agent.vercel.app/widget.js" async></script>

Features

The embedded widget includes:

  • Floating chat bubble with smooth animations
  • Lazy-loaded iframe (only loads when clicked)
  • Mobile responsive design
  • Keyboard accessible (Escape to close)
  • Customizable colors and position
  • JavaScript API for programmatic control

JavaScript API

Control the widget programmatically:

ZingChat.open(); // Open the chat ZingChat.close(); // Close the chat ZingChat.toggle(); // Toggle open/close ZingChat.isOpen(); // Check if open (returns boolean)
